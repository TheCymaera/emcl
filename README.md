# EMCL
EMCL is a language designed to compile to mcfunction files.

## Web Compiler
[URL](https://www.heledron.com/tools/emcl-compiler/)

## Install CLI
```
npm install emcl -g
```

## Run CLI
Create boilerplate project in working directory
```
emcl init
```

Build working directory
```
emcl build
```
